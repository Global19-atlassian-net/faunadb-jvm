package response

Provides representations of the FaunaDB response format.

Most calls to FaunaClient will return a com.faunadb.client.response.ResponseNode. This represents an abstract node in the FaunaDB response tree. These nodes can be converted into basic FaunaDB types, as well as various structured response forms.
